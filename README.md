# An .flp file opener with a version selector that scans installed FL Studio versions.

## Features

- Automatically detects FL Studio installations
- Supports both 64-bit and (legacy) 32-bit FL Studio
- Lets you choose which version opens a project
- Automatically exits after opening FL Studio

## How to use
- Assign file associations to the exe
- Open a .flp and select your version
